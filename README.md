# Create Quick Template

Project template manager and from-template creator.

## Install

```bash
$ npm i -g create-quick-template
```

## Usage

```bash
$ create-quick-template

? What project template would you like to use? (Use arrow keys)
❯ cra-template
  webpack-react
  template3
  ...
```

or

```bash
$ cqt
```
